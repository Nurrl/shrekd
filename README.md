# shrt
SHare and shoRTen, simple file &amp; url sharing service

# Feature ideas
- Shared slugs between redirects and files
- User-programmable:
    - Slug length
    - Persistence time
    - Download count
    - Custom Slug
    - Checksum verification
- Allow pastes
- Delete token
- Retention curve depending on the weight, with expiration in return headers
